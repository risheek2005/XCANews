# 📰 XCANews – SwiftUI News App

A modern iOS news application built using SwiftUI that fetches real-time news using the NewsAPI.

---

## ✨ Features

- 📱 Clean and modern SwiftUI UI
- 🗂 Category-based news (General, Technology, Sports, Business, etc.)
- 🔍 Search functionality
- ⭐ Bookmark articles
- 📰 Featured article section
- 🌐 Live news fetching using NewsAPI
- ⚡ Smooth navigation and animations

## 🛠 Tech Stack

- SwiftUI
- MVVM Architecture
- URLSession (Networking)
- NewsAPI
- Combine / Async-Await

---

## 📂 Project Structure

- Views/ → UI components
- ViewModel/ → Business logic
- Models/ → Data models
- API/ → Network layer
- Assets/ → Images & resources

---

## 🔑 API Setup

To run this project:

1. Get your API key from https://newsapi.org  
2. Open Info.plist  
3. Replace:
      YOUR_API_KEY   
   with your actual API key  

---

## 🚀 How to Run

1. Clone the repository  
2. Open in Xcode  
3. Add your API key  
4. Run on simulator/device
